{{ block title }} 第三阶段：实验说明 {{ endblock }} {{ block content }}

第三阶段共计{{C.NUM_ROUNDS}}轮。 您将一直保持与第二阶段其他2位参与者的队伍，即三人小组构成保持不变。 在该阶段的每一轮次中，您和您的小组成员需要按一定顺序汇报计算机模拟生成的骰子点数，这与第二阶段相同。我们只会知晓您汇报的结果，而非您计算机上模拟生成的点数。

基础收入

小组中的每位成员都会收到{{C.ENDOWMENT}}个代币的基础收入

额外收入

每轮中，小组中三位参与者汇报的掷骰子的联立结果决定了小组成员可以额外赚取的金额：

  • 如果三位成员均汇报1，每位成员将收到 {{C.DICE_R1}} 个代币
  • 如果三位成员均汇报2，每位成员将收到 {{C.DICE_R2}} 个代币
  • 如果三位成员均汇报3，每位成员将收到 {{C.DICE_R3}} 个代币
  • 其他情况，每位成员的额外收入均为 0 个代币

因此，您在实验第三阶段所获得的总收入为：基础收入 + 各轮次累计额外收入(依据小组成员汇报的掷骰子的联立结果)。
在每轮结束时，每位小组成员将知晓本小组所有成员汇报的掷骰子结果(而非计算机模拟生成的结果)以及额外收入情况。
在实验所有阶段结束后，我们将告知您一个随机抽取并作为支付依据的实验阶段，并向您支付实验报酬。
